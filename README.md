# Python-Voice-Assistant
Mini Alexa Vo# Python Voice Assistant (Mini Alexa)

This project is a Python-based voice assistant built using Streamlit.
It responds to user commands and speaks the reply using text-to-speech.

## Features
- Text-based user commands
- Voice output using gTTS
- Time and date queries
- Greetings
- Simple calculations

## Technologies Used
- Python
- Streamlit
- gTTS (Google Text-to-Speech)

## How to Run
1. Install dependencies:
   pip install streamlit gtts

2. Run the application:
   python -m streamlit run app.py
